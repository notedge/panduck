    1. Fenced Code Blocks

    Fenced code blocks begin with three or more tildes and end with a line with at least as many tildes.

    program hello
  implicit none
  print '(a)', 'Hello, world!'
end program hello

    They may also have a lang attribute.

    program hello
  implicit none
  print '(a)', 'Hello, world!'
end program hello

    In Kramdown, one specifies the language identifier as follows:

    program hello
  implicit none
  print '(a)', 'Hello, world!'
end program hello

    or with a Maruku-style inline attribute list

    program hello
  implicit none
  print '(a)', 'Hello, world!'
end program hello
